# 分镜优化与声音设计指南

## 一、分镜优化技巧

### 1.1 分镜图的意义

在 Seedance 2.0 等原生视频大模型时代，分镜图的意义已从流水线铺路转变为精准的外科手术，主要用于解决以下三大痛点：

| 痛点 | 解决方案 |
|------|----------|
| 降低视频抽卡率 | 复杂极限镜头先生成分镜图，再跑图生视频 |
| 残次镜头兜底补救 | 通过"抽帧 → 改图 → 重新图生视频"优化 |
| 空镜头与情绪定格 | 解决视频大模型难以直出的精美静态画面 |

### 1.2 出图改图工具推荐

| 工具 | 核心优势 | 适用场景 |
|------|----------|----------|
| **Nano Banana 系列** | 对中国风元素理解到位，价格便宜 | 角色、场景出图 |
| **GPT Image 2** | 文字渲染能力强，效果惊艳 | 封面、海报、复杂构图 |
| **Seedream系列** | 即梦/豆包图片模型，价格便宜 | 快速出图、风格化处理 |
| **LibTV** | 聚合平台，无需魔法 | 综合出图改图 |

### 1.3 标准出图提示词公式

```
【艺术风格/媒介】+【景别与视角】+【主体描述】+【环境场景】+【光影与色调】+【画质与质感修饰词】
```

**示例：**
```
王家卫电影质感，真人影视剧照。近景特写，45度侧拍。一名30岁的冷峻男子，面容硬朗，穿着黑色定制西装，微微低头，右手把玩着一枚银色打火机。背景是奢华的极简风总裁办公室，落地窗外有模糊的都市夜景。伦勃朗光，冷色调为主，打火机火苗提供暖色补光。真实的皮肤毛孔纹理，浅景深，胶片颗粒感。
```

### 1.4 多资产融合生图

**参考生图的提示词公式：**
```
(风格)+(景别)+(主体描述) + (环境) + (光影) + (质感)
```

**实战案例：角色与场景融合**
```
电影级大片截图，中景镜头（腰部以上构图），侧面拍摄，请将图1中的男主角无缝置入图2的深夜跨海大桥场景中，他正靠在桥边栏杆上抽烟，神情疲惫。霓虹冷色调，桥面的粉蓝色灯光自然映射在男主的侧脸和风衣上。
```

### 1.5 图片快速裂变技巧

#### 1.5.1 多视角转换
```
基于此图，生成XX视角/XX景别
```

**示例：**
```
生成女人的脸部特写，动作、视线的方向保持不变。
```

#### 1.5.2 九宫格批量生成
```
根据这张图，生成画面中女人的不同视角、景别的图，女人的姿势和动作要保持不变，3*3九宫格，16:9的出图比例，左上角要有清晰的数字序号标注。
```

#### 1.5.3 LibTV多角度编辑器
支持六种预设角度：
- 鱼眼视角
- 倾斜视角
- 正面俯拍
- 正面仰拍
- 全景俯拍
- 背面视角

#### 1.5.4 角色站位图
对于复杂室内场景，提前生成人物位置关系图作为参考，确保空间关系清晰。

### 1.6 视频抽帧修改

**操作流程：**
1. 在剪辑软件中抽取崩坏前最后一帧完美画面
2. 利用 GPT Image 2/Nano Banana 结合提示词生成下一个连贯动作画面
3. 以此图作为首帧参考，回到视频模型中重新生成后半段视频
4. 在剪辑软件中进行拼接

**提示词示例：**
```
将男人手握着的物件去掉，手指向前方，其余不变。
```

### 1.7 Seedance 2.0 原生字幕消除

#### 1.7.1 剪映AI消除
1. 使用剪映AI消除功能
2. 选择智能选取，找到字幕最长帧
3. 涂抹识别，点击消除

#### 1.7.2 LibTV智能去字幕
- **智能擦除**：AI自动识别并一键去除，速度较慢
- **框选擦除**：手动框选字幕区域进行擦除，速度较快

## 二、声音设计指南

### 2.1 音乐应用原则

**短剧音乐特点：**
- **去OP/ED化**：短剧很少有长篇大论的片头片尾，通常直接进剧情
- **功能性BGM**：强调在短时间内迅速拉起观众情绪

### 2.2 AI音乐工具推荐

| 工具 | 核心特点 | 备注 |
|------|----------|------|
| **Suno** | 乐器编排、音质清晰度、长段落连贯性优秀 | 需要魔法，免费版有每日限制 |
| **Udio** | 旋律和编曲备受好评 | 需要魔法，免费使用 |
| **网易天音** | 专业级创作平台，编辑自由度高 | 免费额度限制 |
| **海绵音乐** | 完全免费，操作简单 | 适合自媒体创作者 |
| **Tunee** | 对话式音乐创作Agent，功能全面 | 支持MV制作、音轨分离 |
| **Mureka** | 商用级音乐推理大模型，中文演唱出色 | 支持超长时段生成 |
| **豆包** | 完全免费，门槛极低 | 适合快速体验 |

### 2.3 Suno 使用技巧

#### 2.3.1 Simple模式
适合快速生成简单基调的纯音乐：
1. 打开 Instrumental（纯音乐）开关
2. 在描述栏填入风格提示词

#### 2.3.2 Advanced模式
适合精确控制音乐结构：
1. 在 Lyrics 栏输入含有元标签的指令
2. 在 Styles 栏输入曲风提示词

**元标签指令示例：**
```
[Intro]
[Verse]
[Chorus]
[Bridge]
[Outro]
```

### 2.4 剪映音乐素材库
直接在剪映音乐库搜索情绪关键词（紧张、危险、热血等），注意选用可商用素材。

### 2.5 音效设计

**音效三层结构：**
| 层级 | 类型 | 作用 |
|------|------|------|
| 第一层 | 环境音 | 建立空间感 |
| 第二层 | 动作音 | 增加真实感 |
| 第三层 | 特殊音效 | 强调视觉冲击 |

### 2.6 音效获取方法

#### 2.6.1 AI音效生成
- **可灵**：文生音效、视频生音效
- **即梦**：内置音效生成功能
- **剪映AI音效**：右键素材→智能生成AI音效

#### 2.6.2 素材叠加法
- 剪映音效库
- 爱给网（国内最大免费音效库）

### 2.7 配音工具推荐

| 工具 | 核心特点 | 适用场景 |
|------|----------|----------|
| **DubbingX** | 语料丰富，多情绪控制强 | 精细化制作 |
| **MiniMax** | 拟真度高，自带口癖、呼吸声 | 自然对话、英文配音 |
| **即梦** | 克隆声音功能 | 应急使用 |
| **ElevenLabs** | 多语言支持，国际标杆 | 多语言/海外题材 |
| **IndexTTS2/GPT-SoVITS/Qwen-TTS** | 本地开源 | 有技术背景用户 |

### 2.8 获取声音资产流程

1. 将生成的视频导入剪映
2. 找到对应角色说话区域（快捷键I设置起始，O设置终点）
3. 选中不超过5秒钟的说话片段
4. 导出音频作为角色音色资产
5. 后续生成视频时上传此音频作为参考

## 三、素材衔接技巧

### 3.1 抽帧接力
```
将@图片X作为首帧
```

**操作步骤：**
1. 在LibTV中鼠标悬停视频，点击右下角相机图标截取尾帧
2. 或在剪映中提取尾帧
3. 在下一段素材提示词中添加：`@图片X作为首帧`

### 3.2 延长视频
```
将@视频1延长XX秒
```

**注意事项：**
- 此方法抽卡率较高
- 音色还原效果一般
- LibTV需先进行Seedance 2.0合规校验

### 3.3 镜头组接原则
- 确保光影、站位差异不大
- 保持角色一致性
- 注意情绪连贯性

## 四、实用提示词模板

### 4.1 参考生图提示词
```
电影级大片截图，[景别]，[角度]，请将图1中的[角色]无缝置入图2的[场景]中，[动作描述]。[色调描述]。
```

### 4.2 多视角转换提示词
```
基于此图，生成[视角/景别]，[主体]的姿势和动作保持不变。
```

### 4.3 四宫格场景提示词
```
根据上传图生成一张2*2四宫格场景多视角参考图。左上为俯拍视角，右上为正向视角，左下为左侧视角，右下为右侧视角。四格必须表现同一个场景，保持同一空间、同一光照、同一色调和同一氛围。
```

### 4.4 局部修改提示词
```
将[主体]的[部位/物品][修改内容]，其余保持不变。
```

### 4.5 脸部转绘提示词
```
仅将皮肤区域（面部）转绘为彩色素描效果，其他包括服装、发型颜色等一切不改变。
```

### 4.6 脸部加网格提示词
```
真人写实风格。在图中左侧的人脸皮肤区域，覆盖上一个线条密度适中的白色3D几何拓扑线框网格。这个网格应紧密地贴合面部的每一个轮廓和特征。必须严格控制网格的范围，确保它仅覆盖皮肤，而不影响耳朵、头发、眼睛、嘴唇或衣领，不要遮住眼眶。图中右侧所有全身像区域都必须保持完全一致，不被修改。
```

## 五、常见问题与解决方案

| 问题 | 解决方案 |
|------|----------|
| 真人审核限制 | 局部面部转绘、平台站内洗图、脸部加网格、脸部特写遮挡 |
| 多角色生成失败 | 单个角色分别生成4秒视频，通过审核后再合并 |
| 视频质量下降 | 使用抽帧接力而非延长视频功能 |
| 字幕乱码 | 使用剪映AI消除或LibTV智能去字幕 |
| 角色一致性问题 | 提供角色参考图，使用相同提示词风格 |
