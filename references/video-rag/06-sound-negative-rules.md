# Sound And Negative Rules

If the platform can generate sound, sound must be controlled as explicitly as image.

## Silence / Minimal Sound

```text
无台词，无唱腔，无念咒，无锣鼓，无夸张钟声，无自动配乐。
只保留轻微环境声：雨声、风声、衣料摩擦、水声、脚步或禅杖轻触石阶。
如果平台无法稳定控制声音：静音生成，不要自动音效。
```

## Negative Blocks By Failure

### Wrong Action

```text
不要坐下，不要跪下，不要盘腿，不要弯腰坐地，不要转身离开，不要跑，不要跳，不要离开画面。
```

### Camera

```text
不要突然运镜，不要快速变焦，不要镜头环绕，不要飞行镜头，不要剧烈摇晃，不要切镜头。
```

### Identity / Costume

```text
不要换脸，不要服装漂移，不要头饰消失，不要交换角色服装，不要多出人物分身。
```

### Audio

```text
不要怪音效，不要锣鼓点，不要戏曲唱腔，不要夸张钟声，不要恐怖片音效，不要自动 BGM。
```

### Text / Modern

```text
不要字幕，不要可读文字，不要水印，不要 logo，不要现代建筑，不要现代门把手，不要游客。
```
