# Generic Transition Shot

Use for motivated transitions, match cuts, sound bridges, gaze bridges, object bridges, light bridges, action bridges, or moving from one scene to another.

## Purpose

The transition must name the visible or audible bridge. Do not write vague "smooth transition".

## Required Prompt Details

- Identify previous endpoint and next start idea.
- Choose exactly one main bridge: sound, object, gaze, light, occlusion, motion, color, or action.
- State whether this is a transition inside the same space or an independent cut to a new scene.
- If using occlusion, name what fills the frame and when.
- If using sound, name what carries across the cut.
- End on a frame that makes the cut obvious.

## Avoid

- Abstract "cinematic transition" phrasing.
- Multiple bridge devices at once.
- Unmotivated blur, whip pan, glitch, or dissolve unless the style calls for it.
- Changing identity/costume/location before the cut without explanation.

## Useful Structure

```text
00:00-00:02 hold previous endpoint.
00:02-00:05 bridge begins: sound/object/gaze/light/occlusion/action.
00:05-00:08 bridge fills or points the frame toward next clip.
结尾：transition anchor is clear and cuttable.
```
