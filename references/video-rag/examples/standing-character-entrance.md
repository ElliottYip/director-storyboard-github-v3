# Standing Character Entrance

Use for: 出场、入场、缓步、三步、定身、亮相、镇守。

## Required Controls

- State the character remains standing for the full clip.
- Step count and path.
- Prop position if any.
- Endpoint pose.
- Forbid sitting/kneeling/turning away.

## Template

```text
## 人物状态
[角色] 从 [起点] 走到 [终点]，全程站立，情绪 [冷静/威严/警觉]。

## 动作提示词
00:00-00:01 [角色] 在 [起点] 站直，[道具/手] 位于 [位置]。
00:01-00:03 向前走 [N] 步：第一步 [脚/方向]，第二步 [脚/方向]，第三步停在 [终点]。
00:03-00:04 双脚站定，[道具] 保持 [垂直/横向/收于身侧]。
00:04-00:06 [眼神/头部] 从 [方向] 转向 [方向]，身体不移动。
00:06-00:07 保持站立定身姿态，形成可剪切静帧。
明确禁止：不要坐下，不要跪下，不要盘腿，不要弯腰坐地，不要转身离开，不要跑。

## 表演 / 微表情
眼神由低垂到抬起；眉头轻微收紧；嘴巴闭合，不说话；呼吸克制。

## 运镜
固定正面中远景，35mm 历史剧镜头，只有非常缓慢的推近。不要横移、环绕、突然拉近、切镜头。
```

## Example: Fahai Entrance

```text
法海全程站立，绝对不要坐下、不要跪下。00:00-00:01 法海站在山门阴影内，身体正直，禅杖竖在身前，头微低。00:01-00:03 他从门内向前走三步：右脚过门槛，左脚跟上，第三步停在石阶中央。00:03-00:04 双脚站定，禅杖垂直立在身前。00:04-00:06 缓慢抬眼看向江边方向。00:06-00:07 保持站立定身姿态。
```
