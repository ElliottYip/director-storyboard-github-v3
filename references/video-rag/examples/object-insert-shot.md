# Object Insert Shot

Use for clues, props, hands, letters, weapons, phones, cups, keys, ritual objects, product reveals, or physical evidence.

## Purpose

The shot should make one object action legible while protecting hand anatomy, scale, material, and continuity.

## Required Prompt Details

- Name the object, material, scale, and location in frame.
- Define hand/object contact: grip, touch, slide, lift, place, open, close, break, spill.
- Keep the object visually consistent with previous state.
- Use a simple camera: macro close-up, locked insert, slow push, or slight rack focus.
- If text exists, specify unreadable texture unless exact text will be added in post.
- End with the object in a stable final state.

## Avoid

- Extra hands, fused fingers, floating objects, or impossible grip.
- Readable random text.
- Product-like glamour lighting unless the style is advertising.
- Changing the prop's color, size, or damage state unintentionally.

## Useful Structure

```text
00:00-00:02 object already visible and stable.
00:02-00:05 hand enters / contacts object.
00:05-00:08 one clear object change happens.
结尾：object holds in final state, easy to cut.
```
