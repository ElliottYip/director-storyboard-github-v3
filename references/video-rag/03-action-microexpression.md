# Action And Micro-Expression Rules

Do not write literary action only. Write physical path, joints, timing, and endpoint.

## Action Detail Checklist

- Starting pose
- Foot path or body path
- Hand/prop path
- Cloth/water/object reaction
- What remains still
- Endpoint pose
- Bad actions to forbid

## Micro-Expression Checklist

- Eyes: gaze direction and change
- Brows: small tension/release
- Mouth: closed/open, no dialogue if needed
- Breath: restrained, heavy, silent
- Pause: where the actor holds

## Example

```text
法海全程站立，绝对不要坐下、不要跪下、不要盘腿。
00:00-00:01 法海在门内阴影中站直，禅杖竖在身前，头微低。
00:01-00:03 向前走三步：右脚过门槛，左脚跟上，第三步停在石阶中央。
00:03-00:04 双脚站定，禅杖垂直，双手扶杖。
00:04-00:06 眼皮从低垂缓慢抬起，目光看向江边。
00:06-00:07 保持站立定身姿态。
```

## Bad Abstractions

Replace these:

- "亮相" -> exact endpoint pose.
- "缓步出" -> step count and foot path.
- "求见" -> sleeve/hand/gaze posture.
- "法力发动" -> body/cloth/water visible cause-effect.
- "护退" -> which hand supports, where body moves, what blocks view.
