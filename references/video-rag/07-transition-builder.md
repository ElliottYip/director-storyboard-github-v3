# Transition Builder

Every adjacent clip needs a named bridge. Do not write only "smooth transition".

## Bridge Types

| Type | Use |
|---|---|
| sound | bell tail, rain, flood roar, gate thud |
| action | sleeve lift continues, hand lowers, footstep completes |
| gaze | character looks toward next subject/location |
| object | door seam, staff, sleeve, waterline |
| light | dark gate cuts to dark interior, warm highlight match |
| occlusion | sleeve/water/mist/door fills frame |
| water | water rising/receding continues between shots |

## Prompt Form

```text
转场锚点：上一镜的 [visible/audible thing] 延续到下一镜的 [thing]。
结尾保持 1 秒可剪切端点。
```

## Examples

- `钟声尾音接到下一镜门内闷响。`
- `白袖停在半空，接到下一镜白袖展开起水。`
- `水花遮挡前景，擦出下一镜无辜剪影。`
- `青袖扫满画面，切到下一镜暗色湿木山门。`
